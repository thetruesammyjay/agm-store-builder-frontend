# AGM Store Builder - Frontend

A modern, scalable e-commerce store builder platform built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Multi-store management
- Product catalog with variants and inventory tracking
- Real-time order management
- Integrated payment processing (Monnify)
- Analytics dashboard
- Image uploads with Cloudinary
- Responsive design
- Server-side rendering

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **API Client:** Axios
- **Image Handling:** Cloudinary
- **Payment:** Monnify Integration
